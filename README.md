# Buildable Seeds

Public seed prompts and build briefs derived from [Buildable Report](https://github.com/nxlv-ai/buildable) analyses.

Published by [NXLV](https://nxlv.ai). Authored by Lucio Amorim.

---

## What this repository contains

Sanitized **Seed Prompts** and **Build Briefs** derived from public Buildable
Report analyses.

These are AI-native build starting points — not clone instructions. They
describe analogous product concepts and architecture directions that builders
can use as a starting point for their own projects.

---

## What seed prompts are

A seed prompt is a structured, sanitized build brief that translates a product's
public surface anatomy into an AI-native build path.

Use them as:
- A starting point for an AI-native build session
- An analogous product concept for a different market
- An educational reference for understanding how a category of software is structured

Do **not** use them as:
- Instructions to clone or replicate another company's product
- A basis for intellectual property claims
- An endorsement that the source product is weak or replaceable

## Structure

```
/seeds/
  <company-slug>.md     Seed prompt / build brief (narrative)
  <company-slug>.json   Structured build brief (metadata + prompt blocks)

/indexes/
  by-category.md        Grouped by product category
  by-framework-version.md   Grouped by framework version
```

---

## Language standards

All seed prompts use sanitized language:

**Preferred:** `seed prompt`, `build brief`, `analogous build path`,
`AI-native translation`, `starting point`

**Avoided:** `clone`, `copy this app`, `rebuild this company`,
`replace this product`

---

## Eligibility

A seed prompt is published here only after:

1. The corresponding Buildable Report passes editorial review
2. The seed prompt is sanitized (no company-specific traces, no hostile framing)
3. The Lovability Fit band supports an AI-native translation path
